This guide explains how to install the Nuxt AEO module in your Nuxt project.

## Requirements

- `Nuxt 3.x` or higher
- `Node.js 18.x` or higher

## Installation

::code-group
```bash [pnpm]
pnpm add nuxt-aeo
```

```bash [yarn]
yarn add nuxt-aeo
```

```bash [npm]
npm install nuxt-aeo
```

```bash [bun]
bun add nuxt-aeo
```
::

2. Add the module to `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-aeo'],
})
```
